# Route Ownership

- `auth.js`: registration, login, JWT session identity
- `students.js`: student profile CRUD
- `internships.js`: internship listing and recruiter posting endpoints
- `applications.js`: apply flow and recruiter status updates
- `notifications.js`: read/unread notification access
- `recommendations.js`: AI/rule-based ranking endpoint
- `cv.js`: PDF upload and parsing endpoints
